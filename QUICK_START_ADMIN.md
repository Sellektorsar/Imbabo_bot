# ⚡ Быстрый старт: Добавление товаров в админ панели

## 🚀 За 5 минут до первого товара

### 1️⃣ Настройка доступа (1 мин)
```bash
# 1. Скопируйте .env.example в .env
cp .env.example .env

# 2. Откройте .env и замените:
# ADMIN_IDS=ВАШ_TELEGRAM_ID
# BOT_TOKEN=ВАШ_ТОКЕН_БОТА
```

**Как узнать свой Telegram ID:**
- Напишите боту @userinfobot
- Скопируйте ID и вставьте в ADMIN_IDS

### 2️⃣ Запуск бота (1 мин)
```bash
# Установка зависимостей
pip install -r requirements.txt

# Инициализация БД
python init_db.py

# Запуск бота
python main.py
```

### 3️⃣ Создание первой категории (1 мин)
1. Отправьте `/admin` боту в Telegram
2. **📦 Управление каталогом**
3. **➕ Добавить категорию**
4. Название: `🕶️ Солнцезащитные очки`
5. Описание: `Стильные очки для защиты от солнца`

### 4️⃣ Добавление первого товара (2 мин)
1. **➕ Добавить товар**
2. Выберите созданную категорию
3. Заполните поля:

```
Название: Ray-Ban Aviator Classic
Описание: Классические очки-авиаторы с металлической оправой и поляризованными линзами UV400
Метафорическое: Почувствуйте себя пилотом истребителя! Эти легендарные авиаторы превратят каждый ваш выход в триумфальное шествие.
Цена: 15000
Пол: Унисекс
Стиль: Классический
Фото: [отправьте любое фото очков]
```

## ✅ Готово!

- Просматривать каталог (`/каталог`)

## 🔧 Полезные команды

| Команда | Что делает |
|---------|------------|
| `/admin` | Главное меню админа |
| `/stats` | Быстрая статистика |
| `python test_admin.py` | Тест админ функций |
| `python check_setup.py` | Проверка настроек |

## 🆘 Если что-то не работает

1. **Нет доступа к админке:**
   - Проверьте ADMIN_IDS в .env
   - Перезапустите бота

2. **Ошибка при добавлении товара:**
   - Проверьте подключение к БД
   - Запустите `python init_db.py`

3. **Фото не загружается:**
   - Размер должен быть < 20MB
   - Формат: JPG, PNG

4. **Бот не отвечает:**
   - Проверьте BOT_TOKEN в .env
   - Проверьте интернет соединение
