# 🔧 Руководство по использованию админ панели бота @imbabo_bot_v2

## 📋 Как добавить товары в каталог через админ панель

### 🚀 Быстрый старт

1. **Запустите бота** и отправьте команду `/admin`
2. **Убедитесь**, что ваш Telegram ID добавлен в список администраторов в файле `.env`
3. **Следуйте пошаговой инструкции** ниже

### 📂 Шаг 1: Создание категорий

Перед добавлением товаров нужно создать категории:

1. Отправьте `/admin` боту
2. Нажмите **"📦 Управление каталогом"**
3. Нажмите **"➕ Добавить категорию"**
4. Введите название категории (например: "Солнцезащитные очки")
5. Введите описание или отправьте `-` чтобы пропустить
6. Категория создана! ✅

**Примеры категорий:**
- 🕶️ Солнцезащитные очки
- 👓 Имиджевые очки
- 🏃 Спортивные очки
- 👔 Деловые очки

### 📦 Шаг 2: Добавление товара

1. В меню каталога нажмите **"➕ Добавить товар"**
2. **Выберите категорию** из списка
3. **Заполните информацию о товаре** по шагам:

#### 📝 Пошаговое заполнение:

**1. Название товара**
```
Пример: Ray-Ban Aviator Classic
```

**2. Обычное описание**
```
Пример: Классические очки-авиаторы с металлической оправой и поляризованными линзами. Защита UV400.
```

**3. Метафорическое описание** (для создания желания)
```
Пример: Почувствуйте себя пилотом истребителя! Эти очки превратят вас в героя голливудского фильма. Взгляды прохожих будут прикованы к вашему стилю.
```

**4. Цена**
```
Пример: 15000
(только число в рублях)
```

**5. Пол**
- 👨 Мужской
- 👩 Женский  
- 👫 Унисекс

**6. Стиль**
- 😎 Классический
- 🏃 Спортивный
- 👔 Деловой
- 🎨 Модный

**7. Фото товара**
- Отправьте фото товара
- Бот автоматически сохранит его

### ✅ Результат

После заполнения всех полей товар будет создан и появится в каталоге!

## 🛠 Управление существующими товарами

### 📋 Просмотр товаров
1. `/admin` → **"📦 Управление каталогом"** → **"📦 Товары"**
2. Выберите товар из списка
3. Просмотрите детали и управляйте

### ✏️ Редактирование товара
1. Выберите товар в списке
2. Нажмите **"✏️ Редактировать"**
3. Выберите что хотите изменить:
   - 📝 Название
   - 📄 Описание
   - 🎭 Метафорическое описание
   - 💰 Цена
   - 📷 Фото
   - 👤 Пол
   - 🎨 Стиль

### 🔄 Управление статусом
- **🟢 Активировать** - товар виден пользователям
- **🔴 Деактивировать** - товар скрыт от пользователей
- **❌ Удалить** - полное удаление товара

## 📊 Дополнительные функции админ панели

### 📈 Статистика
- Общее количество пользователей
- Количество заказов
- Сумма продаж
- Конверсия по этапам AIDA

### 📧 Рассылки
1. **Создание рассылки:**
   - Введите текст сообщения
   - Добавьте фото (опционально)
   - Выберите аудиторию
   - Отправьте немедленно или запланируйте

### 🎫 Промокоды
1. **Создание промокода:**
   - Введите код (например: FIRST20)
   - Выберите тип скидки (процент/фиксированная сумма)
   - Установите размер скидки
   - Настройте лимиты использования

### 📋 Заказы
- Просмотр всех заказов
- Изменение статусов
- Детали заказа и покупателя

## 🔐 Настройка доступа

### Добавление администратора
В файле `.env` добавьте Telegram ID в `ADMIN_IDS`:
```env
ADMIN_IDS=123456789,987654321,111222333
```

### Получение Telegram ID
1. Напишите боту @userinfobot
2. Скопируйте ваш ID
3. Добавьте в `.env`
4. Перезапустите бота

## 🚨 Важные моменты

### ✅ Рекомендации:
- Используйте качественные фото товаров
- Пишите привлекательные метафорические описания
- Регулярно проверяйте статистику
- Создавайте промокоды для стимулирования продаж

### ⚠️ Ограничения:
- Только администраторы могут управлять каталогом
- Фото должны быть в формате JPG/PNG
- Максимальный размер фото: 20MB
- Цены указываются только в рублях

### 🔧 Устранение проблем:
- **Нет доступа к админке** → Проверьте ADMIN_IDS в .env
- **Ошибка при добавлении товара** → Проверьте подключение к БД
- **Фото не загружается** → Проверьте размер и формат файла

## 📞 Поддержка

При возникновении проблем:
1. Проверьте логи в `logs/bot.log`
2. Убедитесь в правильности настройки `.env`
3. Проверьте подключение к базе данных
4. Обратитесь к разработчику
