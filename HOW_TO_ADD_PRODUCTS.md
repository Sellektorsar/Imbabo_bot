# 🛍️ Как добавить товары в каталог через Telegram бота

## 📋 Быстрый старт (5 минут)

### 1. Настройка доступа администратора

1. **Узнайте свой Telegram ID:**
   - Напишите боту [@userinfobot](https://t.me/userinfobot)
   - Скопируйте ваш ID (например: `123456789`)

2. **Добавьте себя в админы:**
   - Откройте файл `.env` в корне проекта
   - Найдите строку `ADMIN_IDS=`
   - Добавьте ваш ID: `ADMIN_IDS=123456789`
   - Если админов несколько: `ADMIN_IDS=123456789,987654321`

### 2. Запуск бота

```bash
# Перейдите в папку проекта
cd /workspace/imbabo_bot_v2

# Запустите бота
python main.py
```

### 3. Доступ к админ панели

1. Откройте Telegram и найдите вашего бота
2. Напишите команду: `/admin`
3. Вы увидите админ меню с кнопками

## 🏪 Добавление товаров пошагово

### Шаг 1: Создание категории (если нужно)

1. В админ меню нажмите **"📦 Управление каталогом"**
2. Выберите **"📂 Управление категориями"**
3. Нажмите **"➕ Добавить категорию"**
4. Введите название категории (например: "Солнцезащитные очки")
5. Подтвердите создание

### Шаг 2: Добавление товара

1. В меню каталога выберите **"🛍️ Управление товарами"**
2. Нажмите **"➕ Добавить товар"**
3. Следуйте пошаговому процессу:

#### 📂 Выбор категории
- Выберите категорию из списка

#### 🏷️ Название товара
- Введите название (например: "Ray-Ban Aviator Classic")

#### 📝 Описание товара
- Введите обычное описание товара
- Пример: "Классические очки-авиаторы с металлической оправой"

#### ✨ Метафорическое описание
- Введите эмоциональное описание для продаж
- Пример: "Почувствуй себя пилотом истребителя! Эти очки дарят уверенность и стиль настоящего героя неба"

#### 💰 Цена
- Введите цену в рублях (только число)
- Пример: `5990`

#### 👤 Пол
- Выберите: Мужские / Женские / Унисекс

#### 🎨 Стиль
- Выберите: Классический / Спортивный / Модный / Винтажный

#### 📸 Фото товара
- Отправьте фото товара
- Бот автоматически сохранит его

### Шаг 3: Подтверждение

- Проверьте все данные
- Нажмите **"✅ Сохранить товар"**
- Товар добавлен в каталог!

## 🔧 Управление существующими товарами

### Просмотр товаров
1. **"📦 Управление каталогом"** → **"🛍️ Управление товарами"**
2. **"📋 Список товаров"**
3. Выберите товар для просмотра деталей

### Редактирование товара
1. В списке товаров выберите нужный
2. Нажмите **"✏️ Редактировать"**
3. Измените нужные поля
4. Сохраните изменения

### Удаление товара
1. В списке товаров выберите нужный
2. Нажмите **"🗑️ Удалить"**
3. Подтвердите удаление

## 📊 Проверка результата

### Через пользовательский интерфейс:
1. Откройте бота как обычный пользователь
2. Напишите `/каталог`
3. Найдите вашу категорию и товар

### Через админ панель:
1. `/admin` → **"📊 Статистика"**
2. Посмотрите количество товаров в базе

## 🚨 Возможные проблемы и решения

### "У вас нет прав доступа"
- Проверьте, что ваш Telegram ID добавлен в `ADMIN_IDS` в файле `.env`
- Перезапустите бота после изменения `.env`

### "Ошибка базы данных"
- Убедитесь, что PostgreSQL запущен
- Проверьте настройки подключения в `.env`

### "Фото не загружается"
- Отправляйте фото как изображение (не файл)
- Размер фото не должен превышать 20MB

### Бот не отвечает
- Проверьте, что бот запущен: `python main.py`
- Проверьте токен бота в `.env`
